# VisualML

A Python package that simplifies the visualization of multi-dimensional Machine Learning decision boundary models.

## Features

- Works with any Scikit-learn compatible classifier
- 2D projection of high-dimensional boundaries
- Supports SVM, Random Forest, Neural Networks, and more

## Installation

```bash
pip install visualml
```

# Personal Blog & Portfolio

This site itself — a fast, bilingual personal blog and portfolio built with modern web technologies.

## Features

- Bilingual (English / Portuguese) with a language switcher
- Tokyo Night dark theme
- Blog powered by Markdown/MDX Content Collections
- Projects portfolio with tech stack badges
- Responsive design with Tailwind CSS v4
- Deployed on Cloudflare Pages

## Stack

- **Astro 5** — static site generator with Content Collections
- **Tailwind CSS v4** — utility-first CSS via Vite plugin
- **TypeScript** — type-safe components and translations
- **MDX** — interactive blog posts

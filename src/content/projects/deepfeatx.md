# DeepFeatX

A Python package for extracting deep learning features from images using pre-trained Transfer Learning models.

## Supported Models

- ResNet50, ResNet101
- VGG16, VGG19
- EfficientNet variants
- MobileNet

## Installation

```bash
pip install deepfeatx
```
